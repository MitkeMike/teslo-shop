# Teslo API
1. Clone the project
```
2. npm install
```
3. Clone file ```.env.template``` and change the name to ```.env```
4. Change the variables enviroments
5. Run data base 
```
docker-compose up -d
```
6. Run the app
```
npm run start:dev
```
